Herramienta de Ajustes DX (DX Settings Loader)

La herramienta de Ajustes DX "DX Settings Loader (DXSL)" es una aplicación para Xbox 360 que ayuda a guardar y cargar tus ajustes para Rock Band 2 Deluxe. Se necesita porque Rock Band 2 no puede escribir a archivos externos. Source Code
